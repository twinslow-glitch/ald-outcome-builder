# ALD Outcome Builder (Streamlit)

This is a lightweight web form that generates an Academic Language Development outcome.

## Deploy on Streamlit Community Cloud
1. Create a GitHub repo and add these files
   - `app.py`
   - `requirements.txt`
2. Go to https://share.streamlit.io (or Streamlit Community Cloud)
3. Select your repo and set the main file path to `app.py`
4. Deploy

## Run locally
```
pip install -r requirements.txt
streamlit run app.py
```
